== Description ==

Simple to install and get your count post views.

1. Upload plugin to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place <?php echo tamhuyet_count();?> or <?php echo tamhuyet_count($post_id);?> in your templates
